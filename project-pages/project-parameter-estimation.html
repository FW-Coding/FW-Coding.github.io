Parameter Estimation Pipeline

Research Project | Imperial College London ESE Group | 2023–2024

Built a complete parameter estimation workflow that processes raw cycling data from Novonix cyclers, extracts key features, and fits Doyle-Fuller-Newman (DFN) model parameters using customised particle swarm optimisation (PSO). The pipeline achieves approximately 50 mV error in voltage predictions across multiple operating conditions, demonstrating accurate capture of cell behaviour.

Workflow Overview

The pipeline bridges experimental characterisation and physics-based modelling, transforming raw battery test data into validated simulation parameters:

  • Data Ingestion: Automated processing of Novonix cycler files using PyProBE
  • Feature Extraction: Identification of characteristic voltage plateaus, capacity, and resistance
  • Optimisation: Particle swarm optimisation to fit DFN parameters across multiple C-rates
  • Validation: Cross-validation against held-out test cycles to ensure generalisation

Technical Approach

Parameter estimation for physics-based models is challenging due to the high-dimensional parameter space and expensive model evaluations. This project customised PSO to efficiently explore the parameter space while maintaining physical constraints and avoiding local minima.

The pipeline fits transport properties (diffusion coefficients, conductivities), kinetic parameters (exchange current densities), and thermodynamic properties (open-circuit potentials) simultaneously, achieving consistent predictions across discharge rates from 0.1C to 2C.

Open Source Contribution

As part of this work, I contributed column mapping for the Novonix cycler to the open-source PyProBE package, enabling smoother integration of experimental data into battery modelling workflows. This contribution helps other researchers using Novonix equipment to accelerate their parameter estimation workflows.

Tools & Technologies

The pipeline leverages PyBaMM for DFN simulations, PyProBE for data processing, and custom Python scripts for optimisation and visualisation. All code is modular and documented to facilitate future extensions and adaptations.
